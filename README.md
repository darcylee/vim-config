# HowTo

### Just clone this repo

    git clone https://github.com/darcylee/vim-config.git ~/vim-config


### if already have a `.vim` && `.vimrc` in your `Home` dir, remove them or backup

    rm -rf ~/.vim ~/.vimrc

### link new config

    ln -sf ~/vim-config/vimrc ~/.vimrc
    ln -sf ~/vim-config/vim ~/.vim

### start vim

    vim &
